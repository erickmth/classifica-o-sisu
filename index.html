Guia SISU Aluno

Aprenda a visualizar sua classificação no SISU usando o console do navegador. Passo a passo completo para acessar suas informações de ranking.

O que este código faz?

Este código JavaScript busca suas informações de classificação no SISU diretamente da API oficial do sistema. Ele cria uma interface visual dentro do site sisualuno.mec.gov.br para mostrar sua posição em cada curso e modalidade de concorrência.

Nota: O código só funciona quando você está logado no SISU Aluno e acessa seus dados pessoais. Ele não modifica o sistema, apenas exibe informações já disponíveis para você.

Passo 1: Acessar o SISU Aluno

1

Acesse o site oficial do SISU Aluno:

https://sisualuno.mec.gov.br/

Faça login com sua conta Gov.br para acessar suas inscrições.

Passo 2: Abrir Ferramentas do Desenvolvedor

2

Com a página do SISU Aluno aberta, você precisa acessar as ferramentas do desenvolvedor (DevTools) do navegador.

Métodos para abrir:

A

Tecla de atalho (mais rápido):

Pressione F12 no seu teclado.

Ou use Ctrl+Shift+I (Windows/Linux) ou Cmd+Option+I (Mac).

B

Menu do navegador:

Clique no menu (três pontos ou linhas no canto superior direito) → "Mais ferramentas" → "Ferramentas do desenvolvedor".

C

Clique com botão direito:

Clique com o botão direito em qualquer parte da página e selecione "Inspecionar" ou "Inspecionar elemento".

Passo 3: Identificar a Aba Console

As ferramentas do desenvolvedor abrem em uma nova janela ou painel. Você precisa encontrar a aba "Console":

Passo 4: Copiar e Colar o Código

4

Abaixo está o código JavaScript completo. Clique no botão "Copiar Código" para copiá-lo para sua área de transferência:

código-sisu-ranking.js
(async function () { try { const token = sessionStorage.getItem("token"); if (!token) { alert("Token não encontrado no sessionStorage."); return; } const response = await fetch( "https://sisualuno-api.sisualuno.mec.gov.br/api/v1/inscricao/ranking", { headers: { Accept: "application/json, text/plain, */*", Authorization: "Bearer " + token, }, mode: "cors", } ); if (response.status !== 200) { alert(`Houve um erro ao buscar o ranking [${response.status}]`); throw new Error("Failed to fetch"); } const data = await response.json(); // ===== CSS ===== const style = document.createElement("style"); style.innerHTML = ` #ranking-overlay { position: fixed; inset: 0; background: rgba(255,255,255,0.4); backdrop-filter: blur(6px); z-index: 999998; display: flex; align-items: center; justify-content: center; } #ranking-popup { background: #ffffff; width: min(95vw, 900px); max-height: 80vh; border-radius: 16px; box-shadow: 0 20px 50px rgba(0,0,0,0.2); display: flex; flex-direction: column; overflow: hidden; } #ranking-popup-header { padding: 12px 16px; font-weight: 800; font-size: 16px; border-bottom: 1px solid #e5e7eb; display: flex; justify-content: space-between; align-items: center; } #ranking-close { cursor: pointer; font-size: 18px; padding: 4px 10px; border-radius: 8px; } #ranking-close:hover { background: #f1f5f9; } #ranking-popup-content { padding: 14px; overflow-y: auto; display: grid; grid-template-columns: repeat(auto-fill, minmax(260px, 1fr)); gap: 12px; } .ranking-card { background: #f8fafc; border: 1px solid #e5e7eb; border-radius: 12px; padding: 10px; display: flex; flex-direction: column; gap: 6px; } .ranking-course { font-weight: 700; font-size: 14px; color: #0f172a; } .ranking-sub { font-size: 12px; color: #64748b; } .ranking-mod { border-top: 1px dashed #e5e7eb; padding-top: 6px; margin-top: 6px; } .ranking-mod-title { font-size: 11px; font-weight: 700; color: #475569; } .ranking-pos strong { color: #dc2626; font-size: 16px; } #ranking-fab { position: fixed; right: 16px; bottom: 16px; background: #dc2626; color: white; padding: 12px 16px; border-radius: 999px; font-weight: 800; cursor: pointer; z-index: 999997; box-shadow: 0 8px 20px rgba(0,0,0,0.25); } #ranking-fab:hover { background: #b91c1c; } `; document.head.appendChild(style); // ===== OVERLAY + POPUP ===== const overlay = document.createElement("div"); overlay.id = "ranking-overlay"; const popup = document.createElement("div"); popup.id = "ranking-popup"; const header = document.createElement("div"); header.id = "ranking-popup-header"; header.innerHTML = `Classificação no SISU`; const closeBtn = document.createElement("span"); closeBtn.id = "ranking-close"; closeBtn.textContent = "✕"; header.appendChild(closeBtn); const content = document.createElement("div"); content.id = "ranking-popup-content"; popup.appendChild(header); popup.appendChild(content); overlay.appendChild(popup); document.body.appendChild(overlay); // ===== FLOATING BUTTON ===== const fab = document.createElement("div"); fab.id = "ranking-fab"; fab.textContent = "Classificação"; document.body.appendChild(fab); // ===== EVENTS ===== function openPopup() { overlay.style.display = "flex"; } function closePopup() { overlay.style.display = "none"; } closeBtn.onclick = closePopup; fab.onclick = openPopup; // ===== RENDER DATA ===== for (const ranking of data.classificacoes) { const card = document.createElement("div"); card.className = "ranking-card"; const course = document.createElement("div"); course.className = "ranking-course"; course.textContent = ranking.no_curso; const sub = document.createElement("div"); sub.className = "ranking-sub"; sub.textContent = `${ranking.no_ies} • ${ranking.no_campus}`; card.appendChild(course); card.appendChild(sub); for (const modalidade of ranking.modalidades) { const mod = document.createElement("div"); mod.className = "ranking-mod"; const title = document.createElement("div"); title.className = "ranking-mod-title"; title.textContent = modalidade.no_mod_concorrencia; const pos = document.createElement("div"); pos.className = "ranking-pos"; pos.innerHTML = ` Posição ${modalidade.nu_classificacao} de ${modalidade.qt_vagas_modalidade} vagas `; mod.appendChild(title); mod.appendChild(pos); card.appendChild(mod); } content.appendChild(card); } } catch (err) { console.error("Erro ao carregar ranking:", err); } })();
5

Volte para a aba Console no SISU Aluno, clique na área de texto do console e cole o código usando:

Ctrl+V (Windows/Linux) ou Cmd+V (Mac)

Pressione Enter para executar o código.

Passo 5: Usar a Interface

6

Após executar o código, você verá:

  • Um botão vermelho flutuante escrito "Classificação" no canto inferior direito da página do SISU.
  • Clique nesse botão para abrir um painel com suas classificações.
  • O painel mostrará cada curso que você se inscreveu, com sua posição em cada modalidade de concorrência.

Atenção: Este código só funciona enquanto você está na página do SISU Aluno. Se atualizar a página ou sair do site, será necessário executar o código novamente.

Solução de Problemas

?

Problemas comuns e soluções:

1. "Token não encontrado no sessionStorage":

Certifique-se de estar logado no SISU Aluno e na página principal após o login.

2. Código não executa:

Verifique se está na aba Console corretamente. Tente fechar e reabrir as ferramentas do desenvolvedor.

3. Nenhuma informação aparece:

O SISU pode não ter liberado as classificações ainda para sua inscrição.

4. Erro de conexão:

Verifique sua conexão com a internet e tente novamente.

Este guia é apenas para fins educacionais. O código não modifica o sistema SISU, apenas exibe informações já disponíveis.
